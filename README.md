## ⚙️ Como rodar o projeto

### 🔧 Pré-requisitos

* Node.js instalado
* npm ou yarn

---

### ▶️ Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

# Acesse a pasta
cd seu-repositorio

# Instale as dependências
npm install
```

---

### ▶️ Rodando o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```
http://localhost:5173
```
